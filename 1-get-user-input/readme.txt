## Dokumentasi fungsi yang digunakan ##

1. isset
    isset adalah fungsi bawaan php yang bisa gunakan untuk mengecek apakah suatu variabel/array memiliki nilai atau tidak.
    apabila null atau undefined(tidak ditemukan), isset akan mengembalikan boolean false.
    apabila selain null atau undefined, maka akan mengembalikan boolean true.


2. echo
    echo adalah fungsi bawaan dari php yang digunakan untuk menampilkan teks ke layar.
    fungsi ini tidak dapat menampilkan seluruh isi array(akan dijelaskan lebih lanjut di pertemuan 2).


3. concatenasi (penggabungan)
    concatenasi pada PHP berarti penggabungan, concatenasi dilakukan dengan menggunakan tanda titik.
    contoh : 'teks 1' . 'teks 2' akan menghasilkan 'teks 1teks 2' (ingat concatenasi tidak akan menambahkan spasi diantara dua teks tadi, kalau mau ada spasi harus ditambahin manual)
